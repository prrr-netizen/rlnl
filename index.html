디스코드에서 끝내는 라이선스 관리,
rlnl 라이선스 봇.

rlnl은 디스코드 전용 라이선스 관리 봇입니다.
라이선스 발급 · 인증 · 만료 · 역할 연동까지 모두 디스코드 안에서 처리할 수 있도록 목표하고 있습니다. 현재는 테스트 단계로, 전용 테스트 서버에서 먼저 기능을 검증하고 있습니다.

주요 기능

결제 연동 자동 라이선스 발급

결제 완료와 동시에 라이선스 키를 자동 발급하는 구조를 목표로 개발 중입니다.
추후에는 RLNL Mall / 상점과 연동하여, 별도 작업 없이 디스코드에서 바로 키를 발급받을 수 있도록 지원할 예정입니다.

디스코드에서 즉시 라이선스 인증

/license link 명령어 하나로 라이선스를 디스코드 계정 또는 서버에 연결하는 방식을 설계하고 있습니다.
인증 결과는 웹 패널과도 연동되어, 어느 계정/서버에 어떤 라이선스가 연결되어 있는지 한눈에 확인할 수 있게 될 예정입니다.

만료 · 경고 · 역할 자동 관리

라이선스 만료일을 자동으로 체크하고, 디스코드 역할과 연동하여 만료 시 역할 제거 및 알림 발송까지 처리하는 기능을 개발 중입니다.
테스트 서버에서 먼저 안정성을 검증한 뒤, 실제 판매용 서비스에 순차적으로 적용할 계획입니다.

웹 관리 대시보드

브라우저에서 라이선스를 한눈에 관리할 수 있는 대시보드를 준비 중입니다.
라이선스 상태, 사용자/서버 정보, 만료 일정, 사용 로그 등을 직관적으로 확인하고, 필요한 조치를 바로 취할 수 있는 것이 목표입니다.
