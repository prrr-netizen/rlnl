디스코드에서 끝내는 라이선스 관리,
rlnl 라이선스 봇.

rlnl은 디스코드 생태계에 최적화된 라이선스 관리 봇입니다.
결제 후 자동 발급, 계정·서버 단위 인증, 만료·역할 연동까지 라이선스 라이프사이클을 전부 디스코드 안에서 처리하는 것을 목표로 합니다.

주요 기능

결제 연동 자동 라이선스 발급

결제 완료와 동시에 라이선스 키를 자동 발급하는 구조를 목표로 개발 중입니다.
추후에는 RLNL Mall / 상점과 연동하여, 별도 수동 작업 없이 디스코드에서 바로 키를 발급받을 수 있도록 지원할 예정입니다.

디스코드에서 즉시 라이선스 인증

/license link 명령어 하나로 라이선스를 디스코드 계정 또는 서버에 연결하는 방식을 설계하고 있습니다.
어떤 계정/서버에 어떤 라이선스가 연결되어 있는지, 추후 웹 패널에서도 한눈에 확인할 수 있도록 연동할 계획입니다.

만료 · 경고 · 역할 자동 관리

라이선스 만료일을 자동으로 체크하고, 연동된 디스코드 역할을 만료와 동시에 제거하는 기능을 개발 중입니다.
만료 전 알림, 만료 후 안내 메시지까지 단계별로 자동화해 운영 부담을 줄이는 것이 목표입니다.

웹 관리 대시보드

브라우저에서 라이선스를 한눈에 관리할 수 있는 대시보드를 준비 중입니다.
라이선스 상태, 사용자/서버, 만료 일정, 사용 로그를 직관적으로 보여주고, 필요한 조작을 바로 수행할 수 있도록 설계하고 있습니다.
